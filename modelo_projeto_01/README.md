## I. Development Requirements ##

### A. Versions ###

### B. Running Server ###
```bash
$ pip install -r requirements.txt
$ python .
```

### C. Running Tests ###
```bash
$ pip install -r requirements.txt
$ pytest
```

### D. Command to build project in docker ###
```bash
sudo docker build -t mackenzie-group-x .
sudo docker run -p 5001:5001 -d mackenzie-group-x
```
